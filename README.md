# 🌐 Rajendra Margale – Portfolio Website

👉 **Visit Portfolio:**
https://portfolio-ashy-eta-51.vercel.app/

## 📌 Features

* Modern and responsive UI design
* Smooth scrolling and animations
* Interactive project showcase
* Skills and technologies section
* Resume download option
* Lightweight and fast loading

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Three.js (for animations)
* Vercel (for deployment)

---

## 📂 Project Structure

```
portfolio/
│── assets/        # Images, resume, media
│── data/          # Projects and skills data
│── js/            # Animations and logic
│── styles/        # CSS files
│── index.html     # Main entry point
```

---

## ⚙️ Setup & Run Locally

1. Clone the repository:

```
git clone https://github.com/RajendraMargale22/Portfolio/
```

2. Open folder:

```
cd portfolio
```

3. Run:

* Open `index.html` in browser
  or
* Use Live Server (VS Code)
